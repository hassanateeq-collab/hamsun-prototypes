Running locally

Single command to serve everything

Clone the repo, then start a single static server at the umbrella root. All modules are accessible at relative paths matching the GitHub Pages URL structure — same dev experience locally as deployed.

cd hamsun-prototypes python3 -m http.server 4320 # now open http://localhost:4320
